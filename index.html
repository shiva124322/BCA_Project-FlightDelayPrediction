Flight Delay Prediction

Using Aviation Data and Machine Learning

About the Project

This project leverages aviation data and machine learning to predict potential flight delays. The predictive model considers multiple variables like time of year, airlines, and route information to determine the likelihood of a delay.

Live Model Interface

If deployed, the model would be accessible via a Flask app. For now, please refer to the GitHub Repository for source code and model files.

How to Use

  • Clone the repository
  • Install required dependencies
  • Run the Flask app locally
  • Enter flight details and view prediction
